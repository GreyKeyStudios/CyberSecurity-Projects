### Quick Wins (10-15 minutes)

**When you're tired but want to do SOMETHING:**

#### Option 1: IOC Hunting
- [ ] Find 3 suspicious IPs from threat intel feeds
- [ ] Run them through your IOC Helper tool
- [ ] Note patterns (hosting provider, country, etc.)

#### Option 2: MITRE ATT&CK Review
- [ ] Random number generator: Pick 5 technique IDs
- [ ] Read descriptions + examples
- [ ] Write down one detection method for each

#### Option 3: Splunk Practice
- [ ] Pick 3 queries from your cheat sheet
- [ ] Write them from memory (no peeking)
- [ ] Test in Splunk (or just review syntax)

#### Option 4: Phishing Triage
- [ ] Check your spam folder
- [ ] Pick one suspicious email
- [ ] Analyze headers (SPF, DKIM, DMARC)
- [ ] Check sender IP/domain reputation
- [ ] Document verdict

---

### Deeper Practice (25-30 minutes)

**When you have more energy and focus:**

#### Option 1: TryHackMe / LetsDefend
- [ ] Continue current room/path
- [ ] Complete at least one section
- [ ] Document 3 key learnings
- [ ] Add new techniques to your notes

#### Option 2: Write a Casefile
- [ ] Pick a recent lab or create fictional scenario
- [ ] Use your incident report template
- [ ] Include timeline, IOCs, findings
- [ ] Practice writing like you're in a real SOC

#### Option 3: Code Something
- [ ] Run one of your Python scripts
- [ ] Test with real data (logs, IOCs)
- [ ] Modify it to add a feature
- [ ] Document what you learned

#### Option 4: Interview Prep
- [ ] Record yourself answering one STAR question
- [ ] Watch it back (yes, really)
- [ ] Time it (under 2 minutes?)
- [ ] Refine and record again
